Features include:
Custom task creation including skill category selection, goal setting, and the ability to quickly edit tasks anytime.
Experience gain across different categories with detailed statistics.
Streamlined UI allowing the user to spend minimal time logging activities while recieving motivation through experience gain and statistics on their performance to motivate them.
